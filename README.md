# End-to-end Testing

1. Install [Node](https://nodejs.org) 
2. Install NPM Packages 
    ```
    npm i
    ```
3. Run Tests 
    ```
    npm start
    ```
4. Write more tests (spec.js files)
5. Run Tests (#3)
6. Repeat step 4 - 5.
